Welcome to the Winamp 3 Scripting Specification by yours truly, RhinoTrip. See below for Copyright info. Winamp 3 Codename Wasabi is the next generation audio player for Windows, produced by Nullsoft, Inc. Winamp 3 includes extensive support for user-designed graphical user interfaces, or "skins". Much of the functionality of these skins is based upon scripts, which are interpreted by the Winamp 3 scripting engine. The functionality of the engine is almost limitless, and therefore its use is somewhat complicated. This site attempts two things; a) it provides a comprehensive list of the pre-defined classes and functions available to script writers, and b) it provides information and tutorials for those not familiar with WA3 Script programming conventions, and pre-made scripts for those who do not wish to code a WA3 script themselves.

Contributions by: Spoonman, Francis Gastellu

Introduction top  
The Concept
Winamp 3 (WA3) includes extensive support for user-designed graphical user interfaces, or "skins". Much of the functionality of these skins is based upon scripts, which are plain text files written in the WA3 script language and compiled to a binary format that is interpreted by the Winamp 3 Scripting Engine. Through the engine, the scripts can direct many sorts of behaviour for various visible and invisible interface objects such as buttons, faders, and text display areas.

The Script Source
The script source file itself is written in the Winamp 3 scripting language, which is a variant of C++; many common C++ operands such as +, -, ++ and so on function in the WA3 scripting language as they would in C++. If... Else, For, and While looping structures are also present, as are functions and classes just as in any other object oriented programming language - for an explanation of these structures as they pertain to WA3, look at the basic tutorials. The script source files are plain text files, such as those created by notepad, with an .m extension. The .m source file is compiled using mc.exe (included w/ WA3) to a .maki binary file.
