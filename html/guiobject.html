Info
Inherits from:Object
Type:Referenceable, Non - Instantiable
Usage:
Notes:
Objects of class GuiObject are rarely used unless they are created as custom user objects. Otherwise, class GuiObject exists mostly for inheritence purposes, so that visible objects such as buttons can share behaviour without having to re-write lots of code. Any class which inherits from GuiObject can use any of GuiObject's functions at any time.
